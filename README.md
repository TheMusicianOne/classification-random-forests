This is work is the project assignment of Yusuf Ali Hilooglu and Ozgun Mutlu for the course "Data Mining and Machine Learning" in Radboud University.

The source code reads a high-dimensional data (not available within the repo), uses PCA and KMO testing to reduce the dimensionality.

Then, a random forest classifier is tuned and trained.

In order to get the end result and details, one can refer to "Results.pdf"
